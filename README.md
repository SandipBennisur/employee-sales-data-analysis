# Employee Sales Data Analysis

## Overview

This repository contains a sample employee sales dataset created for practicing **Data Analytics** using Python.

The project is designed for beginners and intermediate learners to improve their skills in:

- Pandas
- NumPy
- Matplotlib
- Seaborn
- Exploratory Data Analysis (EDA)
- Data Cleaning
- Data Visualization

---

## Dataset Information

The dataset contains employee-related information including:

| Column | Description |
|---------|-------------|
| Employee_ID | Unique employee ID |
| Name | Employee name |
| Department | Employee department |
| Gender | Male/Female |
| Age | Employee age |
| Experience | Years of experience |
| City | Employee city |
| Sales | Total sales generated |
| Profit | Profit generated |
| Salary | Employee salary |
| Rating | Employee performance rating |
